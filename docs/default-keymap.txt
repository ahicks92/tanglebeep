# Tangledeep Default keyboard layout — snapshot for building the key-evacuation table.
#
# Format: KEY: action, action, ...   (one physical key per line; "Ctrl+" etc. = modifier)
# "(-)" marks a negative axis contribution (the negative half of a UI axis action).
#
# This is the game's "Default" layout (KeyboardControlMaps.DEFAULT), the only layout
# the mod supports. It is the single keyboard ControllerMap the player holds at runtime
# (category "Default", 81 element maps); the MenuControls/Gameplay/UIControls categories
# exist in UserData but are never loaded onto the player.
#
# NOTE: "Cycle Hotbars" (default Ctrl) is intentionally absent — the mod already
# evacuates Ctrl for the screen reader before this snapshot was taken.
#
# Captured 2026-06-19 from the live game via the dev server.
